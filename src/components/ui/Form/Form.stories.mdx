# Form

The `<Form>` component renders the form wrap. The `<Form.FieldGroup>` component wraps a form components with proper spacing,
along with support for a label, info text, and error state. To ensure accessibility, use `<Form.Label>` for the label.

## Fields

All fields are wrapped in `<Form.Field>`.
